High Cloud Airlines Performance Dashboard

Project Overview

This project presents an interactive Power BI dashboard developed to analyze airline performance and operational metrics. The dashboard provides insights into passenger traffic, load factor, flight activity, and overall airline performance.

Tools Used

- Power BI
- Microsoft Excel
- DAX
- Data Modeling

Key Performance Indicators (KPIs)

- Load Factor %
- Total Passengers
- Total Flights
- Weekend vs Weekday Analysis
- Monthly Performance Trends
- Airline-wise Performance Analysis

Dashboard Features

- Interactive Visualizations
- Dynamic Filters and Slicers
- Trend Analysis
- Performance Monitoring
- User-Friendly Dashboard Design

Project Outcomes

This project helped improve skills in data cleaning, data modeling, DAX calculations, KPI development, and dashboard design using Power BI.
